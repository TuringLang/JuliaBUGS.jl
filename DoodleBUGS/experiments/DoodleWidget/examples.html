Quick Start

Add the DoodleBUGS widget to your HTML page:

<!-- Include CSS and JS from dist-lib --> <link rel="stylesheet" href="path/to/doodlebugs.css"> <script type="module" src="path/to/doodlebugs.js"></script> <!-- Use the widget --> <div style="width: 100%; height: 500px;"> <doodle-bugs></doodle-bugs> </div>

Build the widget library using npm run build:lib, then copy dist-lib/doodlebugs.js and dist-lib/doodlebugs.css to your project.
